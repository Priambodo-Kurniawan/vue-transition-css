# vue-transition-css  

## Usage  
include "vue-transition.css" your page  
`<link href="vue-transition.css" rel="stylesheet" type="text/css" media="all" />`

```
<transition name="jump">  
  <div v-if="isShow">Hello World!</div>
</transition>
```


## transition pattern
- jump  
- fade
- slideLeft
- slideRight
- scaleUp
- scaleDown
